# 🛍️ Neural Retail AI Dashboard

## 📌 Project Overview

Neural Retail AI Dashboard is an AI-powered retail analytics platform that helps businesses analyze sales, forecast demand, understand customer behavior, and optimize inventory. The project integrates machine learning models with an interactive Streamlit dashboard and FastAPI backend.

---

## 🚀 Features

- 📊 Interactive Retail Dashboard
- 📈 Demand Forecasting
- 👥 Customer Analytics
- 📦 Inventory Optimization
- ⚡ FastAPI Integration
- 🐳 Docker Support

---

## 🛠️ Technologies Used

- Python
- Streamlit
- FastAPI
- Pandas
- NumPy
- Plotly
- XGBoost
- Docker

## ▶️ Installation

Clone the repository

```bash
git clone https://github.com/ItsPooja-Dev/Neural-Retail-Project.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run Streamlit

```bash
streamlit run app.py
```

Run API

```bash
uvicorn API.main:app --reload
```
